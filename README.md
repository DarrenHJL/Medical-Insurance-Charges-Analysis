# Medical-Insurance-Charges-Analysis

## Overview
This project analyses how demographic characteristics and lifestyle behaviour influences their medical insurance expenses. Instead of a report, the findings are summarised and presented as an interactive slidy presnetation created with R.

## Repository Structure
```
.
├── Analysis.Rmd                            # code
├── header.html
├── Medical Insurance Charges Analysis.pdf  # slidy presentation
└── README.md
```

## Data Availability
The dataset can be found in Kaggle, see references.

## Requirements
- R 4.5.x

## Getting Started
1. Clone the repository.
2. Download the datasets, see references.
3. Run R markdown code
